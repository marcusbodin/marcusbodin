Saturday, January 14th

New adventures ahead!

Although I've enjoyed my time at Combitech I've felt that the work tasks weren't quite for me. Sure it was really cool to work with developing a fighter jet at Saab, but I felt I wasn't really living up to my potential.

Luckily, a new opportunity has come up! Starting February this year I'll be working at Capgemini in Stockholm. My role there will be as a Junior Project Analyst in their graduate program. More specifically I'll be working for Backelite, which is owned by Capgemini. Backelite is a service design agency, which means I'll be working closer to the end user.



Working closer to the end user has been a high priority for me while applying for jobs, so it feels awesome to finally get to do it. Also, the whole IT business seems really fun and interesting so I think I'm going to enjoy it a lot!

It's a bit sad leaving all the fun colleagues I've gotten to know at Combitech and Saab, but hopefully I'll stay in touch with some of them.

Thursday, December 15th

I think I'm getting the hang of this...

...well at least of the HTML and CSS part. I just recently started learning some Java script but I feel like I want to get the site nice and clean before I begin with any cool effects.

The most demanding part has definitely been creating a responsive site. As you'll notice, if you start messing with the browser window size, the content changes with the size of the window. The site is also responsive to smart-phones and tablets to make it more appealing and easier to use.

I think I'll probably take a brake from developing the site for a little while now so that I can focus on studying for the CTP-(Combitech Talent Program) course and the IPMA-certification coming up in January.
